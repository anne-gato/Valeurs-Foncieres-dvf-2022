# Valeurs-Foncieres-dvf-2022
Les prix immobiliers à Paris sont-ils significativement plus élevés que dans le reste de la France?   Ce projet a pour objectif d'analyser le marché immobilier français (Demandes de Valeurs Foncières (DVF)). L'étude se concentre surune comparaison détaillée des prix immobiliers entre Paris et le reste de la France pour l'année 2022.
